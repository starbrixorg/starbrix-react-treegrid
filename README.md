### How to run this app

### Add treegrid licence CODE to `layout.Cfg`

```
// src/layout.ts
{
   Cfg: {
      // ...other props
      Code: <STARBRIX_LICENCE_CODE>
   }
}

```

## Install the dependencies

##### If you are using yarn

```bash
   yarn
```

##### If you are using npm

```bash
   npm install
```

#### Run the app

```bash
   yarn dev
```
